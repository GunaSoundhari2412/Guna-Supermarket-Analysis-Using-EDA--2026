# Exploratory Data Analysis of Supermarket Sales Dataset

## Project Overview

Smart Retail Insights is a data-driven Exploratory Data Analysis (EDA) project developed using Python. This project aims to analyze supermarket sales data to uncover valuable business insights, customer purchasing behavior, sales trends, and performance metrics through statistical analysis and visualization techniques.

The project demonstrates how raw business data can be transformed into meaningful insights that support better decision-making and business growth.

## Project Objectives

- Understand the structure and quality of the dataset
- Perform statistical analysis on sales-related data
- Identify sales patterns and customer preferences
- Analyze payment methods and branch performance
- Visualize business metrics using charts and graphs
- Generate actionable insights from sales data

## Tools and Technologies

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- VS Code
- Git & GitHub

## Dataset Description

The dataset contains supermarket transaction records including:

- Invoice ID
- Branch
- City
- Customer Type
- Gender
- Product Line
- Unit Price
- Quantity
- Sales
- Payment Method
- Customer Rating

The dataset provides a comprehensive view of customer transactions and sales performance across multiple branches.

### Statistical Analysis

### Data Visualization

## Key Findings

- Certain product categories generated significantly higher sales than others.
- Customer purchasing behavior varied across different product lines.
- Popular payment methods were identified through transaction analysis.
- Branch-wise performance comparison highlighted top-performing locations.
- Correlation analysis revealed relationships among numerical variables.
- Customer ratings provided insights into customer satisfaction trends.
